set (3652)


Etymology

Compare the verb settan. Compare Old Norse sæti (whence modern English seat), Old High German gesazi (German Gesäß), Middle Dutch gesaete, from Proto-Germanic *sētiją.

Pronunciation

  • IPA(key): /set/

Noun

set n

  1. seat

Related terms

  • ġeset

Zelfstandig naamwoord

set m [3]

  1. (wiskunde) verzameling (ook: apparaten)
  2. (sport) ronde in een wedstrijd
  3. plaats waar filmopnamen gemaakt worden
